# Real-World Use Case Examples

Complete production templates for common scenarios.

## Use Cases

### 1. SaaS Brand Monitoring
Track mentions of your SaaS product across LLM responses.

**Template**: `saas-brand-monitoring.config.yaml`

### 2. Content Gap Analysis
Identify content opportunities to improve your rankings.

**Template**: `content-gap-analysis.config.yaml`

### 3. Competitive Intelligence
Monitor how competitors are positioned vs your brand.

**Template**: `competitive-intelligence.config.yaml`

### 4. LLM SEO Optimization
Optimize your brand presence in LLM responses.

**Template**: `llm-seo-optimization.config.yaml`

## How to Use These Templates

1. Copy the template config
2. Replace placeholder brands with your actual brands
3. Customize intents for your industry
4. Run daily/weekly for tracking
