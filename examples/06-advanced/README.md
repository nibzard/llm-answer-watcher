# Advanced Examples

Production-ready configurations with advanced features.

## Files

- `budget-controls.config.yaml` - Cost management and limits
- `browser-automation.config.yaml` - Steel browser runners
- `high-concurrency.config.yaml` - Parallel processing at scale
- `production-ready.config.yaml` - Everything combined

## Features Demonstrated

### Budget Controls
- Per-run spending limits
- Per-intent cost caps
- Warning thresholds
- Cost estimation before execution

### Browser Automation
- ChatGPT web interface via Steel
- Perplexity web interface
- Screenshot capture
- Session management

### High Concurrency
- Parallel query execution
- Rate limit management
- Optimized for large-scale monitoring
